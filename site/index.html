What is Zaruka?

A decentralized personal AI assistant that runs entirely on your machine. No data leaves your server.

Model-agnostic: use Claude, GPT, Ollama, or any LLM provider you prefer.

Chat with it through Telegram. Manage tasks, check weather, track email subscriptions, and more.

Features

Everything you need in a personal assistant, nothing you don't.

🔒

Self-hosted & private

All data stays on your machine. No cloud, no tracking, no compromise.

💬

Natural language interface

Chat with your assistant via Telegram, just like messaging a friend.

✅

Task management

Create, track, and complete tasks with smart reminders.

🌦️

Weather & marine

Get forecasts and ocean conditions for any location worldwide.

📧

Email tracking

Monitor subscriptions and important emails automatically.

🔧

Extensible skills

Modular architecture — add new skills as you need them.

How it works

1

Install

Run the install command or deploy with Coolify. Takes under a minute.

2

Configure

Answer a few setup questions: Telegram bot token, AI provider, API key.

3

Chat

Open your Telegram bot and start chatting. That's it.

Deployment options

CLI (npm)

Install globally and run on any machine with Node.js 20+.

curl -fsSL https://zaruka.pro/install | bash zaruka setup zaruka start

Coolify / Docker

One-click deploy with Coolify, or use Docker Compose directly.

git clone https://github.com/user/zaruka # Set env vars in Coolify UI or .env docker compose up -d
